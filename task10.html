CSS

what is CSS?

CSS stands for Cascading style sheets.Cascading style sheet languege used for specifying the presentation and styling of a document written in a mark up languege or XML.

What is purpose of CSS?

The purpose of cascading style sheets is to format web pages by controlling their appearence and layout.

Types of CSS

CSS has 3 Types

Inline CSS

Internal CSS

External CSS

Inline CSS

Inline CSS involves applying styles directly to individual HTML elements using the style attribute.

Internal CSS

An internal CSS is used to define a style for a single HTML page.An internal CSS is defined in the head section of an HTML page, within a style element.

External CSS

External CSS is a form of CSS which is used to add styling to multiple HTML pages at a time.
